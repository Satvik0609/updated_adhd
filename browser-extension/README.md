## Installation

1. Open Chrome/Edge and go to `chrome://extensions/` (or `edge://extensions/`)
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `browser-extension` folder
5. The extension is now installed!

## Usage

1. Click the extension icon in your browser toolbar
2. Add websites to block (e.g., facebook.com, twitter.com)
3. Click "Start Timer" to begin a focus session
4. Blocked sites will be automatically blocked during the session
5. Get a notification when the session completes

## Permissions

- **Storage**: Save blocked sites and timer state
- **Tabs**: Block navigation to distracting sites
- **Alarms**: Schedule timer notifications
- **Notifications**: Show focus session reminders
- **Web Navigation**: Intercept and block site navigation

## Privacy

All data is stored locally in your browser. No data is sent to external servers.
